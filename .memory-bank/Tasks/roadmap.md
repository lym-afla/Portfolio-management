# Roadmap — Suggested Milestones

- **M1 (0–1 month)**: CI setup, critical unit tests, cache invalidation hooks
- **M2 (1–3 months)**: DB migration to Postgres, basic deployment to VPS/VM, secrets management
- **M3 (3–6 months)**: Introduce background worker (Celery), robustness for large imports, canary deploys
- **M4 (6–12 months)**: Productize: UX polish, onboarding flows, GDPR and tax compliance review, performance optimizations
