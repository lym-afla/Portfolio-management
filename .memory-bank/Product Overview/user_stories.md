# User Stories — Portfolio Management App

- **Import transactions**: As an investor I want to import transactions (CSV/API/PDF) so that I can track all my trades and holdings.
- **Dashboard overview**: As an investor I want to see my dashboard (NAV by date, IRR, NAV composition) to understand portfolio performance at a glance.
- **Positions detail**: As an investor I want to see open and closed positions and drill into realized/unrealized gains by security.
- **Multi-currency reporting**: As an investor I want correct multi-currency reporting (select base currency) and explicit FX-effect decomposition.
- **Bond support**: As an investor I want accurate bond handling (price as % of nominal, notional changes, coupon schedules) so I can follow fixed-income instruments correctly.
- **Automated broker sync**: As a power user I want to register broker API tokens and fetch transactions automatically.
- **Export reports**: As an investor I want to export tax and performance reports (CSV/PDF) for accounting and compliance.
